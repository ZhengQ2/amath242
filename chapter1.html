Observations:

  1. Floating point numbers are not eqally spaced. The spacing "jumps" by a factor of 2 at each power of 2.
  2. There is an awkward gap between 0 and the smallest normalized number.
  3. 0 is unrepresentable in this system.

Definition 1.4: The distant from 1 to the next larger normalized floating point number is called the machine epsilon, denoted as ϵmach\epsilon_{mach}ϵmach​.
We have the following,

  • 1=(0.10…00)b×b(0…01)b1 = (0.10\dots00)_b \times b^{(0\dots01)_b}1=(0.10…00)b​×b(0…01)b​

  • ϵmach=(0.00…01)b×b(0…01)b=b1−m\epsilon_{mach} = (0.00\dots01)_b \times b^{(0\dots01)_b} = b^{1-m}ϵmach​=(0.00…01)b​×b(0…01)b​=b1−m

Question: Is floating point addition associative, i.e., x⊕y⊕z=x⊕(y⊕z)x \oplus y \oplus z = x \oplus (y \oplus z)x⊕y⊕z=x⊕(y⊕z)?
Try in MATLAB:
(1 + eps / 3) + eps / 3 == 1 + (eps / 3 + eps / 3)
Answer: False. (1 + eps / 3) + eps / 3 would return 1, while 1 + (eps / 3 + eps / 3) would return 1 + eps.

This problem is significant, see the following example:

while x <= pi
    ...
end

If x is very close to pi, the loop may iterate one more time or one less time than expected, due to the accumulation of errors.

A better way to write the loop is:

while abs(x - pi) > eps(x)
    ...
end

This way, the loop will iterate until x is within the machine epsilon of pi.

§1.2 Conditioning of a math problem

  • Consider a problem PPP with input x⃗\vec{x}x and output y⃗=fP(x⃗)\vec{y} = f_P(\vec{x})y​=fP​(x).
  • No algorithm or floating point system involved yet!

Definition 1.9: (Conditioning of PPP)

  • PPP is well-conditioned with respect to the absolute error if small change in Δx⃗\Delta \vec{x}Δx in x⃗\vec{x}x result in small change in Δz⃗\Delta \vec{z}Δz in z⃗\vec{z}z.
  • PPP is ill-conditioned with respect to the absolute error if small change in Δx⃗\Delta \vec{x}Δx in x⃗\vec{x}x result in large change in Δz⃗\Delta \vec{z}Δz in z⃗\vec{z}z.
  • Similarly, we can define both with respect to the relative error.

Intermission: (Vector norms)
Let x⃗=(x1,x2,⋯ ,xn)∈Rn\vec{x} = (x_1, x_2, \cdots, x_n) \in \mathbb{R}^nx=(x1​,x2​,⋯,xn​)∈Rn.

  • 2-norm: ∥x⃗∥2=x12+x22+⋯+xn2\|\vec{x}\|_2 = \sqrt{x_1^2 + x_2^2 + \cdots + x_n^2}∥x∥2​=x12​+x22​+⋯+xn2​​.
